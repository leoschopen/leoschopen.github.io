1.19

1，如果数组的大小较大，应该将其定义在主函数外面，否则会使程序异常退出，函数内部申请的局部变量来自系统栈，函数外面的全局变量来自静态存储区，允许申请的空间较大

2，字符数组末尾都有一个空字符“\0”表示字符串的结束，gets和scanf都会自动帮你添加，puts和printf通过识别\0来判断结尾输出

但getchar不会自动添加，后面会出现乱码（dev c++中是一个换行,VC6.0是乱码，VC6.0里没有初始化的变量值就是0xcccccccc，对应的中文编码刚好就是“烫”，字符串输出的时候，指针读取超过了本字符数组的结束而不到下一个字符串的开始，指针指向就是这两个中间的位置这些内存char值一般在VC里会是205，而两个字节的205连在一起，正好是汉字“烫”的代码）

3，字符数组可以直接输出cout<<str;

整个字符串一次输入

char c1[5],c2[5];

cin>>c1>>c2;

cout<<c1<<” “<<c2<<endl;

1.20

一、关于结构体在C和C++之间的不同

1，c++中struct能包含成员函数，能继承，能多态

二、那C++中的struct和class有什么区别

最主要的却别是：默认的继承访问权限不同。struct是public的，class是private的。

默认的继承访问权限是看子类到底是用的struct还是class

三、gets，getline(C++)，scanf

1，gets，scanf遇到回车判断停止，gets可以接收空格，scanf不可

2，getline()和cin.getline()类似，但是cin.getline()属于istream流，而getline()属于string流，是不一样的两个函数

getline(cin,str) string

cin.getline(str,100) char array

3，gets=cin.getline()

const double eps = 1e-8

#define Equ(a,b) (fabs((a)-(b))<(eps))

关键是把取值约束在某一范围内
